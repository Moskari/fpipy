========
Glossary
========


.. glossary::

   CFA
     Short for :term:`Colour Filter Array`.

   Colour Filter Array 
     Array of various colour filter used to enable
     measurement of colour images using a monochromatic sensor by sampling
     different colours from neighbouring points. Usual arrangements red, green
     and blue pixels are referred to as Bayer filters.

   DN
     Short for :term:`Digital Number`.

   Digital Number
     Values received from the sensor (after analog-to-digital conversion and
     possible sensor firmware modification).

   Fabry-Perot interferometer
     Optical component composed of two parallel mirrors with a set air gap
     between them. Used as a tunable bandpass filter in hyperspectral imagers.
     See `Wikipedia <https://en.wikipedia.org/wiki/Fabry%E2%80%93P%C3%A9rot_interferometer>`_ for more.
